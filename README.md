# Prediction of Heat Exchanger Outlet Temperature Using Linear Regression

## Problem Statement

Heat exchangers are critical equipment in chemical and process industries. Accurate prediction of outlet temperature helps optimize energy efficiency and process control.

This project develops a Linear Regression model to predict the hot-fluid outlet temperature based on operating parameters such as inlet temperatures and flow rates.

---

## Objective

To build a Linear Regression machine learning model that predicts the hot-fluid outlet temperature using process operating data.

---

## Input Features

- Hot fluid inlet temperature ($T_h$) [K]
- Cold fluid inlet temperature ($T_c$) [K]
- Hot fluid flow rate [kg/s]
- Cold fluid flow rate [kg/s]

---

## Target Variable

- Hot fluid outlet temperature ($T_{h,out}$) [K]

---

## Workflow

1. Generate or collect heat exchanger operating data.
2. Perform Exploratory Data Analysis (EDA).
3. Preprocess the dataset.
4. Train a Linear Regression model.
5. Evaluate model performance using R² Score and RMSE.
6. Analyze feature importance and interpret results using heat transfer principles.
