PROBLEM STATEMENT:
Heat exchangers are critical equipment in chemical and process industries.
Accurate prediction of outlet temperature helps optimize energy efficiency and process control. 
This project develops a linear regression model to predict the hot-fluid outlet temperature based on operating parameters such as inlet temperatures and flow rates

OBJECTIVE:
To build a linear regression machine learning model that predicts the output temperature using process operating data

 INPUT FEATURES:
 - hot fluid inlet temperature(T_h in K)
 - cold fluid inlet temperature(T_c in K)
 - hot fluid inlet rate (kg/s)
 - cold fluid inlet rate (kg/s)

Target variable:
- hot fluid outlet temperature (in K)

Workflow
1.Generate or collect heat exchanger operating data.
2.Perform exploratory data analysis.
3.Preprocess data.
4.Train a Linear Regression model.
5.Evaluate using R² and RMSE.
6.Analyze feature importance and physical interpretation
